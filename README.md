# Stream3D Project Page

Project landing page for Stream3D: Sequential Multi-View 3D Generation via Evidential Memory.

## Files

- `index.html` — single-page project site (inline CSS, no JS)
- `paper.pdf` — submission PDF

## Local preview

```
python3 -m http.server -d . 8000
```
Then open <http://localhost:8000>.

## Hosting

This repository is intended to be served via GitHub Pages from the
`main` branch root. Enable Pages in repository settings if not already
configured.
